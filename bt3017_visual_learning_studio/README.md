# BT3017 Visual Learning Studio

An interactive Streamlit learning platform for selected BT3017 topics.

## Topics Covered

- PCA
- Audio Features
- Graph Learning

## Features

- Guided concept explanations
- Interactive visualisations
- Topic-based quizzes
- Progress and review tracking
- Modular multi-page architecture
- Integrated AI Tutor chatbot
